### 📌 구현기능

---

-   로그인
    -   OAuth (카카오, 네이버)
-   Todo CRUD
    -   Todo 작성, 삭제
    -   Todo 상세 작성, 수정, 삭제
-   Todo History 및 셀프 피드백
    -   이전날들의 작성한 Todo기록
    -   피드백
