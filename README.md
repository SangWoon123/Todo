계획 하고있는 하루의 목표를 기록하는 TodoList 서비스

### ⚒️기술스택

---

-   Front: Vue, Quasar famework, Pinia
-   Backend: Java 17, Spring Boot 3.0, JPA, gradle 2.7, Swagger
-   DB: MySQL
-   Dev-ops: AWS, Github action, Docker

### 📌 구현기능

---

-   로그인
    -   OAuth (카카오, 네이버)
-   Todo CRUD
    -   Todo 작성, 삭제
    -   Todo 상세 작성, 수정, 삭제
-   Todo History 및 셀프 피드백
    -   이전날들의 작성한 Todo기록
    -   피드백
